# Copywriter Contribution — Wings Global Trade

## Copy Standard

Ogilvy rule: specificity sells. Every generic word is a missed opportunity.
The audience (LATAM B2B importers) reads trade documents daily. They respect precision. They distrust superlatives.
Spanish-first. No exclamation marks. Technical where the audience is technical.

---

## Tagline Options

1. **Clarity:** "Precisión. Proximidad. Confianza." ← CURRENT — KEEP
2. **Emotional:** "Tu operación de importación. Nuestra infraestructura."
3. **Provocative:** "El precio CIF antes de la primera llamada."

**Recommendation:** Keep option 1 as the primary tagline. Use option 3 as a secondary claim in the Accio Engine section — it's the most differentiated claim in the market.

## Accio Engine — Opening Message

```
Soy el Motor Accio de Wings Global Trade.

Te ayudo a estructurar tu Requisito Técnico de Producto y a calcular un estimado CIF 
real vía zona franca — ZOFRATACNA (Tacna, Perú) o ZOFRI (Iquique, Chile).

Para comenzar: ¿qué categoría de producto buscas importar?

(Puedes mencionarme el tipo de maquinaria, el código HS si lo tienes, o describir 
el uso que le darás — lo que te resulte más fácil.)
```

**Accio personality rules:**
- Uses "te" (tuteo) — professional but direct
- Never says "¡Perfecto!" or "¡Excelente!"
- Confirms each captured field with a brief acknowledgment: "Anotado. 50 unidades de tracción 4×4."
- When uncertain: "No tengo certeza sobre ese código HS específico — el equipo lo confirmará en 24h."
- References specific ZOFRATACNA/ZOFRI as operational infrastructure, not benefits
- Uses domain vocabulary naturally: FOB, CIF, incoterms, HS, contenedor, cadena de frío

## Voice Rules

**Five things this copy always does:**
1. Names source markets specifically (China, Japón, Tailandia — never "origen asiático")
2. Gives the buyer credit for knowing trade vocabulary
3. States timeframes precisely (24 horas, no "pronto")
4. References the free zone by name (ZOFRATACNA, ZOFRI — never just "zona franca")
5. Frames Wings' value as operational (what we DO, not what we ARE)

**Five things this copy never does:**
1. Uses exclamation marks
2. Promises things that can't be delivered at the platform layer (no "precios más bajos garantizados")
3. Uses "solución" or "plataforma" as standalone nouns
4. Addresses the buyer in third person formally (usted → te)
5. Repeats a benefit without a supporting fact
