# Wings Global Trade — Data Model

## Overview

Supabase Postgres. Row Level Security enabled on every table — no exceptions.
All tables use `uuid` primary keys generated by `gen_random_uuid()`.
All tables include `created_at` and `updated_at` timestamps.
No user authentication in MVP — all writes are from anonymous visitors (server-side via service role key).

---

## Tables

### `categories`

Catalog categories. Seeded at deploy time. Not user-editable in MVP.

```sql
CREATE TABLE categories (
  id          uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  slug        text UNIQUE NOT NULL,           -- 'maquinaria-agricola', 'camiones', etc.
  name_es     text NOT NULL,                  -- Spanish display name
  name_en     text NOT NULL,                  -- English (SEO metadata)
  description_es text,                        -- Optional category description
  icon_key    text,                           -- Maps to local SVG icon set
  sort_order  integer NOT NULL DEFAULT 0,
  is_active   boolean NOT NULL DEFAULT true,
  created_at  timestamptz NOT NULL DEFAULT now(),
  updated_at  timestamptz NOT NULL DEFAULT now()
);
```

**Seed data:**
| slug | name_es | name_en |
|------|---------|---------|
| maquinaria-agricola | Maquinaria Agrícola | Agricultural Machinery |
| camiones | Camiones | Trucks |
| buses | Buses | Buses |
| equipo-industrial | Equipo Industrial | Industrial Equipment |
| repuestos | Repuestos | Spare Parts |

**RLS:**
```sql
ALTER TABLE categories ENABLE ROW LEVEL SECURITY;
-- Public read access
CREATE POLICY "categories_public_read" ON categories
  FOR SELECT USING (is_active = true);
-- No insert/update/delete from client
```

### `products`

Wings curated product catalog. Managed by Wings team directly in Supabase (or via a future admin UI).

```sql
CREATE TABLE products (
  id              uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  category_id     uuid NOT NULL REFERENCES categories(id) ON DELETE RESTRICT,
  slug            text UNIQUE NOT NULL,
  name_es         text NOT NULL,
  name_en         text NOT NULL,
  description_es  text NOT NULL,
  description_en  text,
  specs           jsonb NOT NULL DEFAULT '{}',    -- Key-value pairs: {"Potencia": "75 HP", "Peso": "2400 kg"}
  source_markets  text[] NOT NULL DEFAULT '{}',   -- ['China', 'Thailand']
  images          text[] NOT NULL DEFAULT '{}',   -- Array of public storage URLs
  models          jsonb DEFAULT '[]',             -- Array of {name, specs_override} for variants
  is_active       boolean NOT NULL DEFAULT true,
  sort_order      integer NOT NULL DEFAULT 0,
  meta_title_es   text,                           -- SEO
  meta_desc_es    text,                           -- SEO
  created_at      timestamptz NOT NULL DEFAULT now(),
  updated_at      timestamptz NOT NULL DEFAULT now()
);

CREATE INDEX products_category_id_idx ON products(category_id);
CREATE INDEX products_slug_idx ON products(slug);
CREATE INDEX products_is_active_idx ON products(is_active);
```

**RLS:**
```sql
ALTER TABLE products ENABLE ROW LEVEL SECURITY;
CREATE POLICY "products_public_read" ON products
  FOR SELECT USING (is_active = true);
```

### `leads`

All inquiry submissions from all flows. Single table, discriminated by `flow` column.

```sql
CREATE TYPE lead_flow AS ENUM ('catalog', 'accio', 'contact');
CREATE TYPE lead_status AS ENUM ('new', 'contacted', 'qualified', 'closed_won', 'closed_lost');

CREATE TABLE leads (
  id                  uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  flow                lead_flow NOT NULL,
  status              lead_status NOT NULL DEFAULT 'new',

  -- Contact info (all flows)
  full_name           text NOT NULL,
  company             text,
  email               text NOT NULL,
  phone               text NOT NULL,             -- WhatsApp number with country code
  destination_country text NOT NULL,

  -- Catalog flow fields
  product_id          uuid REFERENCES products(id) ON DELETE SET NULL,
  product_name_snapshot text,                   -- Denormalized — product may change
  quantity            text,                     -- e.g. "5 unidades", "1 contenedor 40HC"
  message             text,

  -- Accio flow fields
  accio_project_id    uuid REFERENCES accio_projects(id) ON DELETE SET NULL,

  -- Notification tracking
  whatsapp_sent_at    timestamptz,
  whatsapp_error      text,
  email_sent_at       timestamptz,
  email_error         text,

  -- Metadata
  source_url          text,                     -- Referring page
  user_agent          text,
  ip_country          text,                     -- From Vercel edge headers
  created_at          timestamptz NOT NULL DEFAULT now(),
  updated_at          timestamptz NOT NULL DEFAULT now()
);

CREATE INDEX leads_flow_idx ON leads(flow);
CREATE INDEX leads_status_idx ON leads(status);
CREATE INDEX leads_created_at_idx ON leads(created_at DESC);
CREATE INDEX leads_email_idx ON leads(email);
```

**RLS:**
```sql
ALTER TABLE leads ENABLE ROW LEVEL SECURITY;
-- No public reads — all reads via service role key (server-side only)
-- No public inserts — all inserts via service role key through API routes
-- This table is ops-only
```

**Note on inserts:** Leads are inserted exclusively by server-side API routes using the Supabase service role key. Client code never touches this table directly.

### `accio_projects`

TPR (Technical Product Requirement) documents generated by the Accio Engine. Each Accio inquiry creates one record.

```sql
CREATE TYPE tpr_completeness AS ENUM ('partial', 'minimum', 'complete');

CREATE TABLE accio_projects (
  id                    uuid PRIMARY KEY DEFAULT gen_random_uuid(),

  -- Core TPR fields
  product_description   text NOT NULL,
  hs_code               text,                   -- AI-inferred, buyer-confirmed
  hs_code_confirmed     boolean NOT NULL DEFAULT false,
  quantity              text NOT NULL,           -- "50 unidades" / "2 contenedores 40HC"
  quantity_units        text,                   -- 'units' | 'containers' | 'kg' | 'tons'
  target_price_usd      numeric(12, 2),
  destination_country   text NOT NULL,
  destination_port      text,
  certifications        text[],                 -- ['CE', 'ISO 9001', 'SENASA']
  tech_specs            jsonb DEFAULT '{}',     -- Product-specific technical specs
  packaging_requirements text,
  delivery_timeline     text,                   -- "60 días", "urgente", etc.

  -- Free zone routing
  free_zone             text,                   -- 'ZOFRATACNA' | 'ZOFRI' — AI-selected based on destination
  source_market         text,                   -- AI-recommended: 'China', 'Thailand', etc.

  -- CIF Estimate
  fob_estimate_usd      numeric(12, 2),
  freight_estimate_usd  numeric(12, 2),
  insurance_estimate_usd numeric(12, 2),
  cif_total_usd         numeric(12, 2),
  duty_rate_pct         numeric(5, 2),
  duty_amount_usd       numeric(12, 2),
  free_zone_savings_pct numeric(5, 2),
  estimate_generated_at timestamptz,
  estimate_disclaimer   text NOT NULL DEFAULT 'Estimación preliminar sujeta a cotización formal.',

  -- TPR completeness
  completeness          tpr_completeness NOT NULL DEFAULT 'partial',
  missing_fields        text[],                 -- Fields still needed

  -- Conversation snapshot
  conversation_turns    integer NOT NULL DEFAULT 0,
  conversation_snapshot jsonb DEFAULT '[]',     -- Last N turns for ops review

  created_at            timestamptz NOT NULL DEFAULT now(),
  updated_at            timestamptz NOT NULL DEFAULT now()
);

CREATE INDEX accio_projects_destination_country_idx ON accio_projects(destination_country);
CREATE INDEX accio_projects_created_at_idx ON accio_projects(created_at DESC);
CREATE INDEX accio_projects_completeness_idx ON accio_projects(completeness);
```

**RLS:**
```sql
ALTER TABLE accio_projects ENABLE ROW LEVEL SECURITY;
-- No public access. Server-side only via service role key.
```

### `notification_log`

Audit log for all outbound notifications. Decoupled from leads for retry tracking.

```sql
CREATE TYPE notification_channel AS ENUM ('whatsapp', 'email');
CREATE TYPE notification_status AS ENUM ('pending', 'sent', 'failed', 'retried');

CREATE TABLE notification_log (
  id              uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  lead_id         uuid NOT NULL REFERENCES leads(id) ON DELETE CASCADE,
  channel         notification_channel NOT NULL,
  status          notification_status NOT NULL DEFAULT 'pending',
  recipient       text NOT NULL,               -- phone number or email address
  payload         jsonb NOT NULL DEFAULT '{}', -- What was sent
  provider_id     text,                        -- Twilio SID or Resend message ID
  error_message   text,
  attempts        integer NOT NULL DEFAULT 0,
  sent_at         timestamptz,
  created_at      timestamptz NOT NULL DEFAULT now()
);

CREATE INDEX notification_log_lead_id_idx ON notification_log(lead_id);
CREATE INDEX notification_log_status_idx ON notification_log(status);
```

**RLS:**
```sql
ALTER TABLE notification_log ENABLE ROW LEVEL SECURITY;
-- No public access.
```

## Triggers

### Auto-update `updated_at`

```sql
CREATE OR REPLACE FUNCTION set_updated_at()
RETURNS TRIGGER AS $$
BEGIN
  NEW.updated_at = now();
  RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER set_categories_updated_at BEFORE UPDATE ON categories
  FOR EACH ROW EXECUTE FUNCTION set_updated_at();

CREATE TRIGGER set_products_updated_at BEFORE UPDATE ON products
  FOR EACH ROW EXECUTE FUNCTION set_updated_at();

CREATE TRIGGER set_leads_updated_at BEFORE UPDATE ON leads
  FOR EACH ROW EXECUTE FUNCTION set_updated_at();

CREATE TRIGGER set_accio_projects_updated_at BEFORE UPDATE ON accio_projects
  FOR EACH ROW EXECUTE FUNCTION set_updated_at();
```
