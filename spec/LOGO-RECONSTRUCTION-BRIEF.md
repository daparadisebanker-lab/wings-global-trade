# Wings Global Trade — Logo Reconstruction Brief
**For external designer engagement. Not an internal task.**

---

## CONTEXT

Wings Global Trade is a B2B trade intelligence platform for Latin American machinery importers. The brand visual thesis: *"Import intelligence should read like a certified document, not a marketplace listing."*

The current logo mark was produced in CorelDRAW with organic paths. It has no documented construction grid, no proportional system, and no variation set. It collapses at small sizes. This brief commissions a geometric reconstruction from first principles.

---

## BRAND IDENTITY

**Name:** Wings Global Trade
**Tagline:** Precisión. Proximidad. Confianza.
**Sector:** B2B heavy machinery and commercial vehicle imports via free trade zones (Peru, Chile)
**Visual register:** Technical authority. Certified document. Not a marketplace.

**References (spirit only — do not copy):**
- Lufthansa mark — geometric precision, aviation, authority, the construction grid is the design
- Maersk wordmark — global trade, direct, no decoration, nothing apologetic
- Lloyd's of London certification marks — seal, authority, the mark as a guarantee
- Swiss International Air Lines — geometric wing construction, precision without coldness

---

## THE EXISTING MARK

The current mark is a wings/bird symbol above the wordmark "WINGS". The concept is right — a bird in flight, the sense of crossing borders and distances. The execution needs to be rebuilt from mathematics.

**Source files:** `BRAND ELEMENTS/wings-logo1 - v2.svg` (most recent version)

## REQUIRED OUTPUT

### 1. Primary mark — geometrically reconstructed

**Construction requirements:**
- Every curve must have a defined, documented radius
- Every proportion must derive from a single base unit (recommend: 8pt grid)
- The wing concept must remain — it is the brand idea
- Simplify if necessary for clean small-scale reproduction (fewer path segments = better behavior at small sizes)
- The mark should feel precise and crafted — not illustrative, not organic
- Mathematical construction document (grid overlay, proportional ratios) delivered alongside the SVG

**Feel:** The mark should suggest: precision instruments, flight, crossing distances. Not speed or aggression — authority and coverage.

### 2. Variation set (all mandatory)

| Variation | Description | Use case |
|-----------|-------------|----------|
| a. Primary horizontal | Mark left + WINGS wordmark right | Default usage, all digital surfaces |
| b. Primary stacked | Mark above + WINGS wordmark below | Square contexts, social profile |
| c. Wordmark only | WINGS lettering, no mark | Tight horizontal spaces, small contexts |
| d. Mark only | Isolated symbol, no wordmark | Favicon, embossing, authentication seals |
| e. Reversed white | White version of all above | Navy backgrounds |
| f. Gold mark | Mark in gold only (not wordmark) | Certification stamps, authentication marks |
| g. Single-color navy | All-navy, no gold | Single-color print, embossing |
| h. Favicon small | 32×32 simplified mark only | Browser favicon, app icons |
| i. Favicon tiny | 16×16 — may need further simplification | Small screen contexts |

### 3. Usage rules document

Deliver alongside the SVG files:
- Minimum size at which each variation is used (in px and mm)
- At what size the mark switches to wordmark-only
- Clear space rule defined with visual diagram
- Approved color combinations (navy on warm white, white on navy, etc.)
- Prohibited uses (distortion, rotation, color substitution, shadow)

---

## WORDMARK

The WINGS lettering should be:
- Set in **Flexo Medium** with manual custom spacing
- OR a bespoke letterform that references Flexo's geometric warmth but is custom-drawn
- Letterforms should be open and legible at all sizes
- No decorative serifs, swashes, or ligatures
- The spacing between letters should be slightly generous — this is not a compressed mark

**Not acceptable:** Script, italic, condensed, or any letterform that would look at home on a consumer brand.

## DELIVERY FORMAT

- SVG for every variation (clean paths, no CorelDRAW artifacts, no embedded bitmaps)
- PDF for print review of each variation
- AI or Figma source file (so proportions and grid are editable)
- Mathematical construction document (grid + ratios) as PDF
- Usage rules as single-page PDF

**Not acceptable:** PNG only, rasterized SVG, files that require CorelDRAW to edit.

## TIMELINE

## WHAT SUCCESS LOOKS LIKE

A designer, developer, or print vendor can open the brand assets and:
1. Know exactly which variation to use in any context
2. Place the logo at any size and have it look correct
3. Reproduce the logo in any medium (screen, print, embossing, signage) without guessing
4. Never need to ask "is this the right version?"

The mark at 16px favicon size should be as intentional as the mark at full page scale.
