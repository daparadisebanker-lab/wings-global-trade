## Preview-authoring learnings (folded from the 2026-07-08 waves)

- The preview page's inherited text color is WARM WHITE (site body is navy):
  light-ground cells need explicit wrappers
  `{ padding: 24, background: '#F8F6F0', color: '#001E50' }`. Components
  using `border-current`/inherited color (Button secondary) adapt to ground —
  compose them on BOTH grounds.
- Framer-motion entrances race the capture screenshot. Two distinct fixes:
  (a) component consults `useReducedMotion`/matchMedia (SpecSheet, FillMeter,
  RFQFlow) → top-of-preview `window.matchMedia` patch (see SpecSheet.tsx);
  (b) component animates UNCONDITIONALLY (Toast.tsx) → the matchMedia patch
  does nothing; use an embedded `<style>` with
  `[role="status"]{opacity:1 !important;transform:none !important}` (see
  ToastProvider.tsx). The harness's frozen `Date.now()` is a red herring —
  rAF/performance.now progress normally; the race is fonts-ready timing.
- "Toast" is NOT a component name: the export is **ToastProvider** (group
  `general`, not `primitives`); `--components Toast` silently no-ops.
- TrustFooter needs NO wide cardMode override (fits the default cell) and no
  motion patch (server organ). Its `logoSrc` prop (site passes
  `/Wings-logo-imagotipo-color.svg`) does not resolve in the preview harness
  OR in the uploaded project — designs must pass their own logo asset;
  covered in conventions.md.

## Re-sync risks

- `packages/ui/.ds-styles.css` is GENERATED by `.design-sync/build-styles.mjs`
  (site Tailwind over globals.css) — always re-run `cfg.buildCmd` before the
  converter when apps/site tokens/globals or packages/ui classNames changed.
- `.design-sync/fonts.css` duplicates the 25 @font-face rules from
  apps/site/src/app/globals.css (with 3 renamed collision-safe flexo copies
  generated into `.design-sync/.cache/fonts-src/` by build-styles.mjs) — if
  the site's font stack changes, this file must be re-derived by hand.
- SITE FINDING (not fixed, faithful-synced): globals.css declares NissanOpti
  400/normal twice (NissanOpti.otf, then flexo-medium.ttf) — the later
  duplicate wins in CSS, so "NissanOpti" text may actually render Flexo
  Medium in production AND in this bundle. Flagged to Muaaz 2026-07-08.
- The faithful `pnpm i --frozen-lockfile` was skipped (concurrent session
  editing root package.json/lockfile) — run it on the next clean-tree sync.
- MisterMark is app-local and enters via componentSrcMap + extraEntries —
  if apps/site moves the file, update both config fields.
